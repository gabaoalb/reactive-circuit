# Reactive-Circuit

Reactive-Circuit is circuit simulation software that allows users to design, simulate, and analyze electronic circuits in a reactive programming environment. It provides a user-friendly interface for creating circuit diagrams and supports real-time simulation and visualization of circuit behavior.

## Features

-   Intuitive drag-and-drop interface for circuit design
-   Real-time simulation of circuit behavior
-   Support for a wide range of electronic components (resistors, capacitors, inductors, transistors, etc.)
-   Visualization tools for voltage, current, and power analysis
-   Export and import circuit designs in various formats
-   Scripting support for advanced simulations and custom components
-   Fully written in TypeScript for type safety and maintainability
